# Aplikacja Webowa "EBoard" napisana z frameworkiem Flask i Bootstrap w języku Python

 - Aplikacja oferuje każdemu użytkownikowi prosty interfejs do wyszukiwania i tworzenia 
postów informujących o nadchodzących wydarzeniach. Posty zawierają: tytuł, lokalizację, godzinę, opis, organizatora. Użytkownicy mogą również komentować posty.

 - Użytkownicy mogą stworzyć konto na stronie.

 - Dodatkowo zaimplementowałem testy jednostkowe(tests.py).
